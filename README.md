# TechFusion Event Website

A modern, responsive website for the TechFusion event featuring a dynamic header, speaker section, event tables, and an interactive contact form. Built using **HTML, CSS, and modern CSS animations**.

---

## Features

- Animated gradient header for a modern welcome effect.
- Clean, card-style sections with hover lift and shadows.
- Speakers section with flex layout and interactive cards.
- Responsive tables for schedules or event details.
- Contact form with focus animations, gradient button, and hover effects.
- Fully responsive design for mobile and desktop.
- Smooth, polished animations to enhance user experience.

---


## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations, Gradients)
- Responsive design principles
